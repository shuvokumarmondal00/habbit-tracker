Set up Google Drive sync
📁 One-time Google Cloud setup (~5 min)
1
Go to console.cloud.google.com → Create a new project (e.g. "Habit Tracker")
2
Enable the Google Drive API under APIs & Services → Library
3
Go to APIs & Services → OAuth consent screen → External → fill app name → add your email as test user
4
Go to Credentials → Create Credentials → OAuth 2.0 Client ID → Web application
5
Add your GitHub Pages URL to Authorized JavaScript origins, e.g.:
https://YOUR_USERNAME.github.io
6
Copy the Client ID and paste it into the sync bar above, then click Connect Drive
Your data is saved as habit-tracker-data.json in your Google Drive root. Only you can access it.
